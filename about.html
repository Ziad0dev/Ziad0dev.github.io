ABOUT THIS SITE

This site is a deliberate throwback to when the web was personal, artisanal, and a little weird. No heavy JS frameworks. Just HTML, CSS, a dash of JavaScript, and an enthusiast's stubborn refusal to let the handcrafted internet vanish.

TECH STACK

  • Static HTML generated by a tiny Node script
  • Zero client-side frameworks
  • RSS feed for old-school aggregation
  • Hosted on GitHub Pages

GOALS

  • Celebrate constraints
  • Encourage source reading over dependency hoarding
  • Provide teaching examples for CSS and semantic structure
