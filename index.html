Rendering Pixelated GIFs Without Blurring

Pixel art often looks blurry on HiDPI displays if the browser smooths it. In this tutorial I show how image-rendering: pixelated; rescues sharp GIF edges. Bonus: how to frame them with faux bevel borders for maximum Windows 95 energy.

Code sample: 

.pixel-art {
  image-rendering: pixelated;
  width: 128px;
  height: 128px;
  border: 2px outset #66aaff;
  background:#002244;
}
